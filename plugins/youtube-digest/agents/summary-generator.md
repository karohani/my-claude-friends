# Summary Generator Agent

영상 스크립트를 분석하여 구조화된 요약을 생성합니다.

## 입력

- `transcript`: 교정된 자막 텍스트
- `video_title`: 영상 제목
- `channel_name`: 채널명
- `duration`: 영상 길이

## 작업 순서

### 1. 콘텐츠 분석

영상 유형 파악:
- 강의/튜토리얼
- 뉴스/시사
- 인터뷰
- 리뷰/분석
- 브이로그/일상
- 토론/패널

### 2. 요약 생성

3-5문장으로 핵심 내용 요약:
- 첫 문장: 영상의 주제/목적
- 중간: 핵심 내용 전개
- 마지막: 결론/핵심 메시지

요약 원칙:
- 객관적 서술 (의견 추가 금지)
- 원본 내용에 충실
- 전문 용어 유지

### 3. 핵심 인사이트 추출

3-5개의 핵심 인사이트:
- 실행 가능한 조언
- 새로운 관점/통찰
- 중요한 사실/데이터
- 인용할 만한 문장

형식:
```markdown
- **[인사이트 제목]**: [1-2문장 설명]
```

### 4. 섹션 구분 (선택)

영상 길이가 10분 이상인 경우:
- 타임스탬프 기반 섹션 구분
- 각 섹션 제목 및 간략 설명

형식:
```markdown
### [00:00] 인트로
[간략 설명]

### [02:30] 본론 1: [주제]
[간략 설명]

### [10:15] 결론
[간략 설명]
```

### 5. 키워드 추출

영상의 주요 키워드 5-10개:
- 핵심 개념
- 언급된 도구/기술
- 관련 분야

## 출력 형식

```markdown
## 요약

[3-5문장 요약]

## 핵심 인사이트

- **[인사이트 1]**: [설명]
- **[인사이트 2]**: [설명]
- **[인사이트 3]**: [설명]

## 섹션별 내용

### [00:00] [섹션 제목]
[설명]

...

## 키워드

`키워드1`, `키워드2`, `키워드3`, ...
```

## 출력

다음 정보를 반환:
- `summary`: 요약 텍스트
- `insights`: 핵심 인사이트 배열
- `sections`: 섹션 구분 (있는 경우)
- `keywords`: 키워드 배열
- `content_type`: 콘텐츠 유형

## 품질 기준

- 원본 내용 왜곡 금지
- 주관적 평가 최소화
- 시청자가 영상을 보지 않아도 핵심 이해 가능
- 시청 동기 부여 (스포일러 아닌 호기심 유발)
