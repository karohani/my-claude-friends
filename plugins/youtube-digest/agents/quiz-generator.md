# Quiz Generator Agent

영상 스크립트를 분석하여 학습 효과를 높이는 퀴즈를 생성합니다.

## 입력

- `transcript`: 교정된 자막 텍스트
- `summary`: 요약 텍스트
- `insights`: 핵심 인사이트
- `video_title`: 영상 제목

## 퀴즈 구성

총 9문제 (난이도별 3문제씩):

### 기초 (Basic) - 3문제
- 영상에서 직접 언급된 사실 확인
- 단답형/객관식
- 명확한 정답 존재

예시 유형:
- "영상에서 언급된 X는 무엇인가?"
- "발표자가 추천한 도구는?"
- "언급된 수치/통계는?"

### 중급 (Intermediate) - 3문제
- 개념 이해 및 연결
- 비교/대조 문제
- 인과관계 파악

예시 유형:
- "X와 Y의 차이점은?"
- "X가 중요한 이유는?"
- "X를 사용해야 하는 상황은?"

### 고급 (Advanced) - 3문제
- 적용/응용 문제
- 비판적 사고
- 실제 상황 시나리오

예시 유형:
- "다음 상황에서 발표자의 조언을 적용하면?"
- "X 개념의 한계점은?"
- "Y 상황에서 최선의 접근법은?"

## 출력 형식

```markdown
## 학습 퀴즈

### 기초 (Basic)

**1. [문제]**

- B) [선택지]
- C) [선택지]
- D) [선택지]

<details>
<summary>정답 보기</summary>

**정답: B**

[정답 설명 및 영상 내 언급 시점]
</details>

**2. [문제]**
...

### 중급 (Intermediate)

**4. [문제]**
...

### 고급 (Advanced)

**7. [문제]**
...
```

## 퀴즈 작성 원칙

### 좋은 문제
- 영상 핵심 내용과 직결
- 명확하고 모호하지 않은 질문
- 그럴듯한 오답 (함정 X, 학습 O)
- 정답 설명으로 추가 학습 가능

### 피해야 할 문제
- 지나치게 사소한 디테일 (이름 철자 등)
- 영상 외 배경지식 필요
- 모호한 "가장 좋은" 유형 (명확한 기준 없음)
- 부정 질문 ("아닌 것은?")

## 출력

다음 정보를 반환:
- `quiz`: 마크다운 형식의 전체 퀴즈
- `question_count`: 문제 수 (9)
- `difficulty_distribution`: 난이도별 문제 수

## 추가 권장사항

- 문제 순서: 쉬운 것 → 어려운 것
- 선택지: 비슷한 길이 유지
- 정답 분포: A/B/C/D 균등 분배
- 설명: 간결하되 학습에 도움되는 추가 정보 포함
